## Range Rover Sample
A responsive sample range rover site for practicing HTML, CSS and Semantic-ui

## Build setup

```
# install dependencies
yarn install

# serve with hot reload at localhost:3000
gulp server

```
## Tech stack
- Gulp
- babel
- sass
- semantic-ui
- enquire.js
