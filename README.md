## Download

```
git clone git@github.com:NanJiang327/IFE2018.git
```

## Install

```
cd LearnNH/Websample
yarn install

```

## Semantic

```
gulp build-ui
```

## Watch Semantic

```
gulp watch-ui
```

## Local Server
```
gulp serve
```
